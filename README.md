# Facção Central — FAC1 ⚔️

Página de alistamento do clã **Facção Central** (Clash of Clans), em estilo de diálogo do jogo:

1. **Bárbaro** dá as boas-vindas
2. **P.E.K.K.A** resume as regras de guerra
3. **Mago** pede o cadastro (Nick, WhatsApp e CV)
4. **Bárbaro** agradece — e o resto chega no grupo do WhatsApp

O cadastro é enviado direto para o Google Forms da liderança (POST no `formResponse`).

Também tem o [PDF de recrutamento](Recrutamento-Clash-of-Clans.pdf) com QR Code.

Site estático — basta servir a pasta (Vercel).
